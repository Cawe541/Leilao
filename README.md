# Leilão

## Tecnologias 
* Laravel
* Vue.js
* Tailwind CSS
* Inertia.js
* Vite

## Requisitos
* Autenticação de usuário
* CRUD de leilões
* CRUD de lances
* Listagem de leilões
* Listagem de lances
* Ranking de lances por leilão
* Ranking de lances por usuário
* Leilões publicos e privados
* Leilões com data de inicio e fim
* Leilões com valor mínimo de lance
